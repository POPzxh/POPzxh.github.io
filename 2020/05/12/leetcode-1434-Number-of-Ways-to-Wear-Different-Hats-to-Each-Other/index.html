leetcode 1434. Number of Ways to Wear Different Hats to Each Other

2020-05-12
leetcode
Word count: 887   |   Reading time≈ 5 min

There are n people and 40 types of hats labeled from 1 to 40.

Given a list of list of integers hats, where hats[i] is a list of all hats preferred by the i-th person.

Return the number of ways that the n people wear different hats to each other.

Since the answer may be too large, return it modulo 10^9 + 7.

方法一

我们想到的第一个想法是，我们将像问题一样直截了当：返回n个人戴着不同帽子的方式的数量。我们将为n个人分配不同的帽子，并使用dp计算出路数。

这个方法==TLE==，原因很显然帽子的数量太多了，所以我们需要给帽子分配人

方法二

dp[i][j] := 前i顶帽子的分配情况，j是bit mask dp[3][3]

e.g. dp[3][101] == # of ways using first 3 hats that people 1 and 3 are wearing hats.

Time: O(40 * 2^n * n), where n is the number of people, n <= 10
Explain: There are total hat*assignedPeople = 40*2^n states in dfs(..hat, assignedPeople...) function, each state needs a loop up to n times (int p : h2p[hat]) to calculate the result.

Bottom-UP DP

显然，复杂度为 40*2^n*n
